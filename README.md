# Tazebao-DIR
WordPress plugin that provides the infrastructure, functionality, and metadata needed to manage a digital archive of one or more museum collections.

# tazebao semantic repository plugin

## description

this repository contains a wordpress plugin developed within the project “tazebao – cultural heritage active innovation for sustainable society” (pe00000020, cup j33c22002850006).

the plugin implements the core semantic data model of the tazebao digital repository, providing structured content types, metadata, taxonomies and api endpoints for the management and publication of cultural heritage collections.

the plugin is designed to be reusable and adaptable in different institutional and museological contexts.

## scientific context

the plugin was developed to support research and experimentation in the fields of:
- digital museology
- semantic description of cultural heritage
- accessible and inclusive digital archives

it has been tested in the context of the university museums of Turin, with particular reference to the art brut collections of the "Cesare Lombroso" Museum of Criminal Anthropology and the Museum of Anthropology and Ethnography.

## main features

the plugin provides:

- custom content types for cultural heritage entities (e.g. digital items, collections, museums, resources)
- semantic relationships between entities
- controlled vocabularies and taxonomies
- metadata structure
- rest api endpoints for external applications and interfaces

the plugin acts as the backend foundation for multiple web-based and interactive applications connected to the tazebao ecosystem.

## technical requirements

- wordpress (open source cms)
- php 8.x
- mysql / mariadb
- standard wordpress rest api

no proprietary services are required.

## reuse and extensibility

the plugin is released as open source to encourage:
- reuse in other cultural heritage projects
- adaptation to different collections or institutions
- long-term sustainability of the data model

the architecture allows integration with custom themes, web apps and interactive installations via api.

## relation to other components

this plugin represents the core repository layer of the tazebao ecosystem. frontend interfaces, themes and applications are developed as separate components and consume data exposed by this plugin.
